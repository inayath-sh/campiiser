Once anaconda is installed you can install Neuron as follows 
<Tabs syncKey="OS">
  <TabItem label="Windows" icon="seti:windows" >
    <Steps>

1. [Download](https://github.com/neuronsimulator/nrn/releases/download/8.2.7/nrn-8.2.7.w64-mingw-py-39-310-311-312-313-setup.exe) 
and install using the GUI installer

2. Further help in installation can be found here:
    [Windows Instructions](https://neuron.yale.edu/ftp/neuron/nrn_mswin_install.pdf)



</Steps>
  </TabItem>
  <TabItem label="MacOS" icon="apple">
    <Steps>

1. Install XQuartz: https://www.xquartz.org/


2. Install Neuron using pip

    This should be done in the vscode terminal after camp2025 environment is activated
    ```bash
    pip3 install neuron
    ```


</Steps>
  </TabItem>
    <TabItem label="Linux" icon="linux">
    <Steps>

1. Install Neuron using pip

    This should be done in the vscode terminal after camp2025 environment is activated
    ```bash
    pip3 install neuron
    ```


</Steps>
  </TabItem>
</Tabs>
